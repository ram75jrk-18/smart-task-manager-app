# Task Management System - Final Submission Project

A simple full-stack task management application created for final-week project submission.

## Implemented Requirements
- Role-based access control (Admin and Member)
- Advanced features: notifications and form validations
- Deployment-ready Node.js project structure
- Documentation and presentation included separately

## Demo Credentials
- Admin: admin@demo.com / admin123
- Member: member@demo.com / member123

## Features
- Login authentication
- Admin can create tasks
- Admin and assigned member can update task status
- Members can only view their own tasks
- Client-side and server-side validation
- Notification messages for success and error actions

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Deployment Target: Vercel / Render / Railway

## Run Locally
```bash
npm install
npm start
```
Then open `http://localhost:3000`
